# Sample Job Description - Software Engineer

## Company: TechCorp Innovation Labs

### Position: Senior Full Stack Developer

**Location:** Remote / Bangalore, India  
**Employment Type:** Full-time

---

## About the Role

We are looking for a passionate Senior Full Stack Developer to join our growing engineering team. You will be responsible for building scalable web applications and contributing to our AI-powered products.

---

## Required Qualifications

- **Experience:** 4+ years of professional software development experience
- **Education:** Bachelor's degree in Computer Science or related field

### Technical Skills (Must Have)

- **Frontend:** React.js, Next.js, TypeScript, HTML5, CSS3
- **Backend:** Node.js, Express.js, Python
- **Databases:** PostgreSQL, MongoDB, Redis
- **Cloud:** AWS or Google Cloud Platform
- **DevOps:** Docker, Kubernetes, CI/CD pipelines
- **Version Control:** Git, GitHub

### Preferred Skills (Good to Have)

- Experience with AI/ML integrations
- Knowledge of GraphQL
- Experience with microservices architecture
- Familiarity with Agile/Scrum methodologies
- Open source contributions

---

## Key Responsibilities

1. Design, develop, and maintain full-stack web applications
2. Collaborate with product managers and designers
3. Write clean, maintainable, and well-documented code
4. Participate in code reviews and mentor junior developers
5. Optimize applications for maximum performance
6. Implement security best practices

---

## What We Offer

- Competitive salary: INR 25-40 LPA
- Flexible working hours
- Health insurance
- Learning & development budget
- Stock options
